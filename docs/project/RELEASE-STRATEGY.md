# Release Strategy

## Early releases
- 0.1.x scaffold and docs
- 0.2.x lexer and parser core
- 0.3.x interpreter subset
- 0.4.x formatter alpha
- 0.5.x semantic analysis
- 0.6.x compiler prototype

## Compatibility policy
Until 1.0, grammar and semantics may change as the implementation stabilizes.
Each such change must be documented and versioned in release notes.
