# Project Introduction

Aster aims to combine:
- the readability and directness of Python
- explicit control and systems reach associated with C and Zig
- compositional expressiveness inspired by Lisp-family design

The project is intentionally documentation-led and implementation-grounded.
The repository should evolve in lockstep across:
- language reference
- grammar
- examples
- tests
- toolchain implementation

This starter repository is the foundation for that process.
