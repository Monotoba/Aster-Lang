# Program 06: Generics and Traits (Prototype)

Features used: `typealias`, generic type params, trait decls, `impl Trait for Type` blocks.

Important: traits/impls are currently **compile-time only** (prototype checking). There is no runtime
dispatch or method-call resolution yet.

Run:

```bash
uv run aster run tutorials/programs/06-generics-and-traits/main.aster
```

Type-check (recommended for this program):

```bash
uv run aster check tutorials/programs/06-generics-and-traits/main.aster
```
