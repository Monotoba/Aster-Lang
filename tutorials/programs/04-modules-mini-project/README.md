# Program 04: Mini Modules Project

Features used: `use`, `pub`, multi-file modules, a small “library”.

Run (from anywhere):

```bash
uv run aster run tutorials/programs/04-modules-mini-project/main.aster
```

You can also type-check without running:

```bash
uv run aster check tutorials/programs/04-modules-mini-project/main.aster
```
